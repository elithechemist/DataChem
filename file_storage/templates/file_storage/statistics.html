ChemStats Goals

  1. Maximizing Efficiency
    • Workflow centralized in a single location with a plug-and-play interface
    • Repetative task such as manually relabeling atomic indices are drastically simplified
  2. Minimizing Barrier to Entry
    • New lab members can easily pick up the data science workflow with little to no guidance
    • New labs can begin using the data science workflow without the need for specialized tools and training
  3. Statistical Analysis Transparency
    • Because data is analyzed at the same source, we can have confidence that the methods used are similar
    • Reviewers and readers can view the raw data on ChemStats.org and see how the fit was obtained
  4. Sharing Data
    • With increased usage, the archive of molecular parameters will grow making it easy to perform quick analyses
    • Web-based interface makes it easy to share data with collaborators
    • The utility of the website will grow over time with increased usage
  5. Modularity
    • The Django framework means any python programs can be easily integrated into the website
    • This will increase the impact factor of new statistical methods and parameters as it is trivial to try them with this web interface
